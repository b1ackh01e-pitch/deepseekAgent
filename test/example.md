## Как запустить приложение

1. Установите зависимости: `npm install react react-dom convert-units`
2. Запустите dev-сервер: `npm start`
3. Откройте http://localhost:3000 в браузере

## Как использовать
1. Выберите тип измерения (длина, объем, вес)
2. Введите значение в исходной единице
3. Приложение автоматически покажет результат в выбранной целевой единице
